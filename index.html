About the Workshop

Finding UART and Getting a Shell

Event Time and Location:
  • Date: Saturday, July 20th, 2024
  • Time: 7:00 PM - 10:00 PM
  • Location: Tech Hub, 123 Main St, Anytown, USA
What we will do

In this workshop we will explore the PCB, identify different components, and show how to locate different interfaces and test to see if its UART. A multimeter will be used so if you already have one bring it! Dont have multimetere? Thats ok! We will have some on hand to share. After we find UART get ready to solder! We will solder on pin headers so we can access that sweet sweet shell. We will then use tools like a hydrabus, bus pirate, or a FTDI to connect to the UART get a shell, and haxor the device! After we get a shell we will discuss different methods for finding vulrabilities and how to exploit them. So buckle up, grab your soldering iron, and get ready to haxor some IoT devices!

Here are a list of skills you can add to your tool belt:
  • How to identify different components on a PCB
  • How to find UART on a PCB
  • How to solder pin headers to access UART
  • How to connect to UART using different tools
  • How to get a shell.
